# 角色設定集 Prompt 萬用模板

**用途**：新增 TDT 主持人 / 來賓 / 客串角色時、套用此模板生成角色設定集（character design sheet），確保所有角色採同一套版面 + 美術風格。

**搭配**：生完設定集 → 用 [`char_3q/PROMPT_BRIEF.md`](char_3q/PROMPT_BRIEF.md) 同款結構出 emotion sprite。

---

## 萬用 Prompt 模板

```
角色設定集，白色背景，版面分為 5 區，繁體中文標籤。

整體風格：2D 插畫、動漫寫實風、線條乾淨、色彩飽和、不要像素風。

【角色設定】
{國籍/族裔}{性別}，{年齡區間}，{髮型描述}，
服裝：{上衣描述（顏色 + 款式 + 細節）}、{下身描述}、{鞋款描述}，
體型{體型描述}，臉型{臉型描述}，眼神{眼神 / 氣質描述}。

【版面配置】

1. 主角全身立繪（左側大圖）
   - 全身正面站立，自然姿勢，雙手輕放身側，{站姿微調}

2. 表情設定區（7 種表情，上半身特寫）
   標籤：idle / talk / smile / thinking / surprised / skeptical / wave

3. 姿勢設定區（6 種全身姿勢）
   - 自然站立
   - 單手指向前方演說
   - 雙手交叉胸前
   - 雙手張開熱情說話
   - 握拳強調
   - 側身回頭

4. 背面圖（側背面 + 正背面）

5. 細節特寫（5 格）
   - 臉部正面
   - 髮型側面
   - {上衣關鍵細節}
   - 手部握拳
   - {鞋款細節}

版面標題用繁體中文，整體參考角色設定集排版風格。

將長寬比設為 1:1
```

## 後續流程（每個新角色都跑一次）

1. **填模板生角色設定集** → 存 `assets/char_{slug}/角色設定集.png`
2. **拷貝 `char_3q/PROMPT_BRIEF.md`** → 改名 `char_{slug}/PROMPT_BRIEF.md`
3. **照 PROMPT_BRIEF 出 17 個 emotion** → 存 `assets/char_{slug}/emo_*.png`
4. **跑 chromakey + histogram normalize** → 去綠 + 色彩對齊
5. **接入 server.py / OfficeScene.js** → 時段制 host rotation 邏輯
