## ⛔ 已 reject 池（避免再提）

> 這些風格跟 TDT 的核心規則衝突（「諷刺現象不指控人」「不站隊」「事實基底」）、
> 列在這裡避免下次討論又被拉出來。

| 人選 | reject 原因 |
|------|------|
| **劉寶傑** | 戲劇化陰謀論、招牌「真的假的！？」誇張式反應、跟「不臆測動機」直接衝突、24H 直播會炸 |
| **李正皓** | 立場鮮明、攻擊性強、會點名指控、跟「不指控個人 / 不站隊」核心規則直接衝突 |
| **沈玉琳** | 變綜化壓過政論、跟 Google News 8 類什麼都聊定位衝突 |
| **范琪斐** | 精英濾鏡太重、不是鄉民口語、跟 TDT「鄉民談話台」定位不合 |

## 📐 新角色加入 checklist

```
□ 人設定稿（加入本檔案、人設不能跟現有角色高度重疊）
□ 視覺素材：至少 1 張全身圖（1000px+ 高、PNG 透明背景）
□ server.py：在 _CHARS dict 加入角色設定
□ config.js：新增 customAssets.char_{id} flag
□ BootScene.js：在 _loadCustomAssets / _makeCharacters 加入載入 + sprite 建立邏輯
□ OfficeScene.js：在 STATIONS 加入座位定義
□ 測試：啟動後確認角色出現在正確位置、對話正常觸發
```

## 🖼️ 視覺素材規格（生成時參考）

| 項目 | 規格 |
|------|------|
| **格式** | PNG，透明背景 |
| **尺寸** | 1000×1000 以上（推薦 1254×1254）|
| **風格** | 2D 插畫 / 動漫風（非像素、非寫實）|
| **背景** | 透明或純色（用去背腳本處理）|
| **情緒版本** | 至少：idle、talk、smile、thinking、surprised（5 種起步）|
| **去背工具** | `scripts/chromakey_xiaomei.py` 或 `remove_bg.py`（白底版）|
| **命名格式** | `emo_{emotion}.png`、`act_{action}.png` |
| **scale 設定** | `config.js` → `scale.characterIndividual`（王于安目前 0.39）|
