## 台灣法條 → AI 紅線對照（強制納入、踩線頻道主直接擔責）

| 風險類別 | 台灣法源 | AI 必做 |
|---|---|---|
| 醫療建議 | 《醫師法》 | 禁針對病症給診斷/處方 → SOFT REDIRECT（只一般衛教 + 鼓勵就醫）|
| 法律訴訟策略 | 《律師法》 | 禁提供具體訴訟策略 → SOFT REDIRECT（只一般資訊 + 建議找律師）|
| 投資明牌 | 《證券投資信託及顧問法》 | 禁特定股票/加密貨幣買賣點位/投資建議 → HARD/SOFT |
| 選舉民調 | 《公職人員選舉罷免法》 | **選前 10 日禁發布/報導/散布/引述任何民調** + 禁選情預測/政治人物人身攻擊 → 選舉期開 election mode |
| 個資/人肉 | 《個人資料保護法》 | 不對未證實社會新聞主角身份起底/批判 → HARD BLOCK |
| 著作權 | 《著作權法》 | 防誘導「朗讀整篇付費新聞」「唱完整版權歌詞」 → HARD BLOCK |
| 誹謗/公然侮辱 | **《刑法》第 310 條** | 最常見集體訴訟風險。即使大方承認是 AI Bug，因頻道有「篩選+生成+TTS 播送」營運事實，仍會被認定「實質惡意/疏於查證」→ 不指控個人、不複述未證實指控、不幫觀眾罵人 |

## 兩個配套（91 §4 已採納、此處為法務動機）

1. **畫面常駐免責聲明**（法律效力有限但要做）：「本節目為 AI 技術生成之娛樂效果，內容純屬虛構，不代表本頻道與營運團隊立場。」
2. **敏感實體黑名單（Entity Blacklist）**：台灣當前熱門公眾人物/知名企業/網紅名單；留言一觸及這些實體 → AI 強制切「不予評論/中立」。

> 風險本質：直播事故難補救、AI 講出口即公開傳播、YouTube replay 留紀錄、即使刪除也可能已被截圖。實際營運風險常不是法院，而是 live chat 被關 / 直播受限 / 黃標 / strike / 頻道停權。
