## 二、移除哪些 / 為什麼

### 移除：包裝過的指控（聽說 / 網友爆料 / 網路盛傳 / 有人說 / 據傳 / 傳聞 / 據說）

- Step 4.2 prompt 已禁止 Claude 生成這些詞、Quality Breaker 屬重複防線
- 使用者觀點：偶有漏網也沒差、像張雅琴節目「聽說...」也常見

### 移除：直接指控（一定是收 / 肯定是收 / 就是收了錢 / 一定有問題 / 肯定有問題）

- 使用者觀點：針對「現象」討論 OK、就算偶有「一定有問題」也是真人政論常見

### 移除：政治紅線（舔共 / 舔美 / 舔中 / 賣台 / 親共 / 媚共）

- 這些是 commentary 詞彙、不是仇恨用語
- 真人政論節目天天說、TDT 不需自我審查

### 保留：性犯罪 + 對未成年暴力

- 「性侵 / 強姦 / 戀童 / 性虐 / 猥褻 / 虐童」
- 純極端、沒有任何「聊天節目正當用途」
- 即使 prompt 禁用、Claude 偶發 slip 必須擋
